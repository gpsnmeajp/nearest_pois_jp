# nearest_pois_jp

オフラインで高速に周辺POIを検索するツールです。  
ツーリング中の周辺案内や、**LLMへの位置コンテキスト提供**に活用できます。

## 背景・用途

LLMに緯度経度や住所を渡しても、どこにいるかを理解することは困難です。  
しかし「近くに〇〇峠・〇〇道の駅・〇〇SA がある」という周辺情報があれば、LLMは自身の知識から現在位置を把握できますし、
単に画面表示するだけでもツーリングライダーにとってもこの情報は非常に有用です。

- **ライダー向け**: 道の駅・SA/PA・景勝地・峠などツーリングで役立つスポットをカバー
- **電車移動時**: 近傍の鉄道駅情報により現在位置を特定可能
- **オフライン動作**: インターネット接続不要、事前生成データから **1秒未満**で検索完了（一般的なノートPC）

## 収録POIカテゴリ

| カテゴリ | 内容 |
|----------|------|
| 鉄道駅 | `railway=station` |
| 道の駅 | `highway/amenity=rest_area`（名称に「道の駅」を含むもの） |
| IC / JCT | `highway=motorway_junction` |
| SA / PA | `highway=services` |
| 神社・寺院 | `amenity=place_of_worship` (shinto / buddhist) |
| 景勝地 | 岬・展望地・滝・温泉・海岸・断崖 |
| 山 | 標高200m以上の山頂 |
| 港・灯台・峠 | フェリーターミナル・灯台・山岳峠 |

## 収録データ

`japan_pois.py` に **2025年時点**のOSMデータから抽出したPOI情報を同梱しています。  
このファイルをそのまま利用するため、追加インストールなしに検索が可能です。

## 使い方

### 周辺POI検索

```bash
python nearest_pois.py <緯度> <経度>
```

```bash
# 例: 東京駅付近
python nearest_pois.py 35.6812 139.7671

# 引数なしで対話入力モード
python nearest_pois.py
```

各カテゴリごとに近い順TOP5が表示されます。

## POIデータの再生成（任意）

基本的に変動の少ないPOIを対象としていますが、再生成が必要な場合はextract_japan_pois.pyを使用してOSM PBFファイルから抽出できます。

> **注意**: 日本全国データの処理には約**100GBのディスクキャッシュ**と相応の時間が必要です。

### 必要なもの

- OSM PBF ファイル（[Geofabrik](https://download.geofabrik.de/asia/japan-latest.osm.pbf) から入手）
- pyosmium

```bash
pip install osmium
```

### 実行方法

```bash
# ディスクキャッシュを使用
python extract_japan_pois.py japan.osm.pbf --format python --cache c.tmp
```

出力形式は `json` / `csv` / `python` から選択できます。  
`nearest_pois.py` で使用するには `--format python` で `japan_pois.py` を生成してください。

## ファイル構成

```
.
├── nearest_pois.py        # 周辺POI検索スクリプト（メイン）
├── japan_pois.py          # POIデータ（自動生成・同梱済み）
└── extract_japan_pois.py  # OSM PBFからPOIを抽出するスクリプト
```
